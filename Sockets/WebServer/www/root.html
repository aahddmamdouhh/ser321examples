Simple Web Server

You can make the following GET requests:

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=3&num2=4 -- multiplies the two numbers and returns the result
  • /github?query=users/amehlhase316/repos -- returns GitHub repositories information
